This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Overview

Dank Memer Journal is an application intended to be used as a personal tool for tracking market transactions in the Currency game from the Dank Memer Discord bot.

## Tech Stack

This is a full-stack application built with the following technologies:

- [Next.js](https://nextjs.org)
- [React 19](https://react.dev)
- [Tailwind CSS](https://tailwindcss.com)
- [Shadcn UI](https://ui.shadcn.com)
- [Prisma ORM](https://www.prisma.io/orm)
- [PostgreSQL](https://www.postgresql.org)
- [Neon](https://www.neon.com)
